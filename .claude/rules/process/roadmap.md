## 🔒 Phase 4-B: 매치 분석 모드 [Gate Review 후]

> **Phase 2 완료 후 Gate Review 통과 시에만 착수.**

### 계획된 기능
- 🗺️ **자기장 분석 탭**: 사용자 버튼 클릭 → 스크린샷 1장 → 추천 이동 경로 표시
- ✈️ **낙하 위치 추천**: 비행 경로 스크린샷 → 추천 낙하 지점 3개 표시

### Phase 4-B 전용 HARD LIMITS
**허용**:
- ✅ 사용자 버튼 클릭 시에만 1장 캡처
- ✅ 분석 결과는 별도 독립 창에만 표시

**금지**:
- ❌ 실시간/주기적 자동 캡처 (타이머 금지)
- ❌ 게임 창 위 오버레이
- ❌ 시계열 분석 (여러 스크린샷 엮기)
- ❌ 특정 화면 감지 시 자동 트리거
- ❌ 5초 내 연속 캡처

### 구조적 방어 의무
- 캡처 함수는 반드시 버튼 이벤트 핸들러에서만 호출
- 타이머 기반 캡처 호출 코드 정적 분석으로도 검출 안 되어야 함
- 모든 캡처 시점 감사 로그 기록
- 5초 내 연속 요청은 레이트 리미터로 거부
